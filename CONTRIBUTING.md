# Contributing

## Local checks

Install dependencies:

```bash
npm ci
```

Build TypeScript:

```bash
npm run build
```

Run tests:

```bash
npm test
```

## Style

- Keep changes small and focused.
- Prefer clear, explicit TypeScript over clever shortcuts.
- Keep CLI output user-friendly and consistent.

## PR checklist

- [ ] README updated if behavior changed
- [ ] CHANGELOG updated
- [ ] `npm run build` passes
- [ ] `npm test` passes
